About

This game was created by Omer Maimon and Tomer Weitzman. the challenges we faced in creating this game are mostly related to orchestration all the different entities of the board to work together and react appropriately when encountering a different mob or a different functionality. To handle this issue we had to implement an abstract design that allows us to separate the game flow from the functionality of the specific mobs. The jQuery Plugin used in this assignment are the validation plugin for the registration.
